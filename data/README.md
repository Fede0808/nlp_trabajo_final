# Datos locales (no versionados)

Esta carpeta contiene datasets grandes usados para entrenar y evaluar el proyecto.

- `entrenamiento.csv` (Properati) y `venta_descripcion.csv` (Zonaprop) viven aca localmente.
- Por sanidad de git, todo `data/*` esta ignorado en `.gitignore` y NO se sube al repo.

Si clonas el repo en otra maquina, copia los CSV a `data/` antes de ejecutar notebooks/scripts.
